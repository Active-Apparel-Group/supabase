# DBA Onboarding & Scorecard

This document merges all onboarding and scorecard content for DBAs working on the Supabase garment tracking project.

1. Review the [Workflow Overview](../integration/WORKFLOW-OVERVIEW.md)
2. Study the [PIM Schema](../schema/pim-schema.md) and [OPS Schema](../schema/ops-schema.md)
3. Read the [Migration & Function Index](../migration/MIGRATION_FUNCTION_INDEX.md)
4. Understand the [API Reference](../api/edge-functions.md) and [Webhooks](../api/webhooks.md)
5. Complete the [Centralized Testing Plan](../testing/CENTRALIZED-TESTING-PLAN.md)
